# Changelog

## [0.1.0] - 2026-04-02

### Added
- Initial release with 13 tools across 6 API families
- REST API: Layer search, feature identification, attribute search, feature details
- Geocoding: Address geocoding, reverse geocoding
- Height: Point height, elevation profile
- STAC: Geodata catalog search, collection details
- WMTS: Map URL generation
- ÖREB: Property ID (EGRID) lookup, cadastral extract
